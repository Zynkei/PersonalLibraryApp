# Personal Library App

## Overview
The Personal Library App allows users to manage their personal library by searching for books, adding them to their collection or wishlist, and managing entries with notes. This application interfaces with the Open Library API to retrieve book information.

## Features
- Search for books by title, author, or ISBN
- View and manage personal library and wishlist
- Add notes to book entries
- Duplicate book handling before performing Open Library searches

## Requirements
- Python 3.x
- SQLite

## Installation
1. Clone the repository: `git clone <repository-url>`
2. Change into the app directory: `cd Personal_Library_App`
3. Install necessary dependencies as needed.

## Usage
Run the application with the command: `python main.py`.
